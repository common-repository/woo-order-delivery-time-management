Plugin allows users to choose the order delivery type, order delivery date and, order delivery time on checkout page.
== Description ==
plugin allows users to choices the order delivery type, order delivery date and, order delivery time on checkout page. From admin side store owner can setup the date rang and start time – end time for store pickup or deliver the order.
== Installation ==
1. Log in to your WordPress admin panel and go to Plugins -> Add New
2. Type **Setup Order Delivery Date & Time for Woo Commerce Order** in the search box and click on search button.
3. Find Order Delivery Date & Time Management for Woo Commerce plugin.
4. Then click on Install Now after that activate the plugin.
OR
1. Download and save the **WooCommerce Order Delivery Time Management** plugin to your hard disk.
2. Login to your WordPress and go to the Add Plugins page.
3. Click Upload Plugin button to upload the zip.
4. Click Install Now to install and activate the plugin.

== Frequently Asked Questions ==

= Why use Order Delivery Date & Time Management for Woo Commerce? =
1. A delivery type option, delivery date picker and, estimated delivery period are integrated into the checkout process smoothly.
2. Besides the type, date and, the customer can select an hour that will depend on your configuration of daily shipments and shipping method selected.
3. Specify dates where deliveries are not available, e.g. holidays or other events.
4. Admin views with the chosen delivery date.

= What is setting page name? =

1. Once you installed and activated the plugin, then you can see the **Woo Order Delivery Settings** menu item under the admin slide menu.
2. On that page you can setup **Delivery Date** and **Delivery Time** for all orders.

== Screenshots ==
